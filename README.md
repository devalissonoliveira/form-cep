# FORM CEP
Este projeto faz parte da atividade solicitada pelo professor Kelvys Moura, visando colocar em prática todos os conhecimentos adquiridos de HTML5, CSS3 e JS durante o curso de Full Stack Live da Digital College.

**Objetivo:** Criar um Formulário que preenche as demais informações ao colocar o CEP, utilizando componentização.
